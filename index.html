Perlynx — Quick Guide

What is Perlynx?

Perlynx is a procedural tile texture editor and generator. Design seamless Perlin-noise textures, add overlay shapes, apply SVG masks, build tile libraries with the carousel, and export a game-ready manifest — all from a single HTML file that runs offline.

Left Control Panel

⚙
Settings Set canvas Resolution X/Y (12–2000 px), Zoom the view, pick a UI Theme, or toggle No canvas shadow. Use Download to save a PNG, and Export / Open / Import to save or restore a full JSON snapshot of all tiles and settings.
◈
Mask Clip the canvas to an SVG shape. Pick individual SVGs with 📄, or load an entire folder with 📁 (the folder is remembered between sessions). Use ✎ Edit to open the live mask editor: draw circles, rectangles or lines directly on the canvas, then hit Save or Save As New.
∿
Perlin Noise Scale — zoom of the noise pattern. Octaves — layers of detail (1–8). Persistence — how much each octave contributes. Power — contrast curve. Blur — post-process softening. Pixelation — chunky pixel effect. Seed — deterministic random state. Invert colors — flips the noise output. Use Randomize to scramble the seed.
◐
Colors The noise output is mapped through a gradient built from your colour swatches (dark noise → first colour, bright noise → last colour). Add Color / Remove Color to extend or shorten the palette. Click any swatch to open the colour picker.
◇
Overlays Toggle Enable to scatter decorative shapes over the texture. Choose a shape type (Diamond, Cube, Orb, Star Crystal, Custom), then tweak Count, Size, Facets, Depth, Opacity, Rotation, and spawn Area. Hit Scatter to re-randomise positions.

Interactive canvas: click an overlay shape to select it — then drag to move, grab the outer ring to resize, or the corner handle to rotate.
#
Properties Attach typed metadata to each tile for use in game manifests. Add Bool, Int, Float, or String fields. Values are stored per tile and included in Export files.
💡 The left panel is draggable — grab any empty area to reposition it. Click the ∨ collapse button to shrink it to a thumbnail, and click again to restore.

Right Material Panel

▣
Material (Custom Panel) A secondary noise layer composited over the tile as a surface material. Enable Perlin Noise to unlock its own Scale, Octaves, Persistence, Power, Blur, Pixelation, Seed, and colour palette. The panel is draggable — grab its header to reposition it.

Canvas & Workspace

⊕
Zoom & Pan Hold Ctrl and scroll the mouse wheel to zoom the entire workspace in/out around the cursor. On touch screens, use a two-finger pinch to zoom. The canvas zoom slider in Settings scales only the main tile canvas.
⊞
Spawned Tiles Drag any tile preview from the carousel onto the canvas area to spawn it as a floating copy. Drag the spawned tile near the main canvas edges or corners — it will snap (blue outline) to align perfectly. Click its ✕ button to remove it. On touch, drag to reposition — tap the tile to reveal the ✕ button.

Tile Carousel (Bottom)

▤
Managing Tiles The bar at the bottom holds your tile library. Click a tile to load its settings. Hit + to add a new tile. Hover (or tap on touch) a tile to reveal the ✕ badge — click it to delete (requires at least one tile). At 7+ tiles the bar scrolls horizontally.
↕
Spawning & Drag Drag a tile preview upward/onto the canvas to spawn a floating rendered copy. Spawned tiles help you preview how tiles tile together before exporting.
↺
Reset Checkboxes Reset state on new tile — when checked, clicking + starts from default noise settings. Reset properties on new tile — clears the Properties list for the new tile. Uncheck both to clone the current tile's settings.

Export, Open & Import

↓
Export Saves all current tiles to a perlynx.json file — noise parameters, colours, overlays, mask data, properties, and optional texture images. Use this as your project file or game asset manifest.
⊡
Open Opens a perlynx.json file and replaces all current tiles with the ones in the file. Use this to resume a saved session.
↑
Import Opens a perlynx.json file and appends its tiles after your current ones, leaving existing work intact. Use this to merge tile sets.
💡 The mask folder you last picked is automatically remembered between sessions via the browser's storage — so your SVG library will reload on every visit without any extra clicks.

JS API — perlynx.js
perlynx.js is a standalone library that renders tiles entirely off-screen — no UI required. Include it in any page, or call it from Node (with a Canvas implementation such as node-canvas).
// generateTiles(tiles, imageKey?, thumbnailKey?) const tiles = [ { name: 'grass', walkable: true }, { name: 'water', walkable: false, perlynx: { inputs: { seed: '99', resX: '256', resY: '256' }, colors: ['#0a2a50', '#1a5078'] } } ]; generateTiles(tiles).then(result => { // result[0].texture → full-res PNG data URL // result[0].thumbnail → 48×48 PNG data URL console.log(result); });
Each tile's perlynx object mirrors the structure in the Export JSON produced by this app — paste it directly. Omit perlynx entirely to use all defaults (256×256, ground-like brown noise).
Parameters (all optional, defaults shown):
inputs.resX / resY 256 output size in pixels inputs.seed 12345 noise seed inputs.scale 40 noise scale inputs.octaves 6 fractal octaves inputs.persistence 0.8 octave amplitude falloff inputs.power 1 contrast curve inputs.blur 0 Gaussian blur radius inputs.pixelation 1 pixel-art block size inputs.invert false invert lightness inputs.shapesEnabled false enable mineral overlays + full overlay params (shapeType, shapeCount, …) colors ['#582b13','#63321d'] base gradient stops cmColors ['#888','#aaa',…] overlay gradient stops mineralsArray [] pre-placed mineral positions mineralScatterSeed 0 seed offset for placement
